# 使用方法
1. 貼入 CSS 檔至 snippets 資料夾內，並在 Appearance 設定中啟用
2. 使用 `- [1] Lorem ipsum dolor sit amet, consectetur adipiscing elit.` 格式來套用
	- 1 ~ 9 為 `| list` 樣式
	- 大寫字母為 `== list ==` 樣式，為顏色首字母，使用大寫避免跟主題所提供的 alternative checkbox 衝突

可自行修改要在 `- [ ]` 內輸入的內容及所代表的顏色或格式

# 其他說明
- 第一節定義顏色變數
- 第二節移除 checked box 的樣式 (仍然會有可點擊的方形)
- 第三節將 checked box 移出可視範圍
- 第四節定義 `li` 的 margin 是為了不要讓他們黏在一起

# Note
- 如果一份檔案內只會有一個連續的 list 則可以不需要額外定義 `> p > input:checked`
- markdown 僅允許 `- [ ]` 內輸入一個字元
- 使用 `transform: translateX();` 可以基於原本位置進行相對移動，如果直接修改 margin 需要先知道數值才方便移動到想要的位置
