> 因為 obsidian 會依據滾動位置 載入/刪除 一定範圍外的 tag，所以 fixed banner 會在滾動到存放該 image 的容器被刪掉時消失。
> 所以 vertical 版本僅適用短篇幅文件。

# 使用方法
1. 貼入 CSS 檔至 snippets 資料夾內，並在 Appearance 設定中啟用
2. 在 yaml 定義 cssClass 欄位，並填入 banner / banner-left / banner-right 其中之一，另外如果使用 banner，可以再使用 banner-top / banner-bottom 調整圖片位置 (object-position 預設為 center)
3. 在要設定為banner的圖片後面加上 `|banner` (e.g. `![[image.jpg|banner]]`)


提供 style setting 參數，可使用 style setting 設定 banner 的高度/寬度

# 其他說明
- 如果想套用的格式只有一個，不需要隨檔案調整，可以調整 CSS，保留想要的格式的程式碼，並移除前面的 class 
e.g. 我只想要 top + center align 的 banner，那麼就刪掉除了 `.banner` 開頭以外的，接著拿掉 `.banner` 標籤即可)，這樣不需要用 cssClass 定義

+ 透明遮罩的效果可以透果調整 mask-image 來達到想要的效果


- 最後一節針對跟 banner 重疊時可能會有影響的物件進行調整
	- callout (格式為 minimal theme，其他主題需要自行調整): 因標題有背景，移除後會看到重疊的裝飾框線，固將標題移至框線內
	- codeblock: 含背景，故移除
	- base table: 表格的 head 和 body 皆有背景，故移除。 base card 如有需要移除背景可以添加 `.bases-cards-item`
