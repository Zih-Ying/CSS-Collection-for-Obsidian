# Custom callout
## 使用方法
在 `> [!defined_name]` 中輸入在 CSS 中定義的文字

## 其他說明
- icon 可以至 [Lucide](https://lucide.dev/icons/) 中尋找，使用 `lucide-name` 套用，其中 `name` 為 icon 名稱
- 修改 `--callout-color` 改變顏色，值為 RGB

## Note
- CSS 中定義使用小寫字母，在 Obsidian 中可以使用大小寫

# Grid callout
參考來源為 [Fancy-a-Story](https://github.com/ElsaTam/obsidian-fancy-a-story) 是個很漂亮的主題

## 使用方法
1. 輸入 `> [!grid|type]`，其中 `type` 包含 `col-2`, `col-3`, `col-4`, `25%`, `33%`, `75%` (`%`為兩欄中左側那欄所佔比例，可於程式碼中自行修改)
2. 同一欄的內容前方使用 `>>`，分隔使用 `>`，範例如下

```
> [!grid|col-2]
>> ![[5f0a3d6faff5f190eec77cc7598bbdde.jpg]]
>
>> ![[6e8cb77d1ad84abc420831cb70a76745.jpg]]
```

## 其他說明
- 可搭配 Templater plugin 快速插入，template 內容見 Callout templates.md
- Templater 快速插入快捷鍵預設為 Alt + E
